5. Inline Formaating Text:
	a) Long Question	<blockquote>		= biasa digunakan untuk konten yg bersifat sebuah kutipan
	b) Preformatted Text	<pre>			= digunakkan untukk menampilkan konten sesuai apa yang ditulis tanpa mengilangkan spasi ataupun enter
	c) Figure 		<figure> | <figcaption>	= digunaakan untuk merepresentasikan konten tersendiri
	d) Anchor 		<a>			= digunakan untuk membuat sebuah hyperlink ke suatu halaman atau website lain
		- atribut wejib ada : href="alamat link yg ingin dituju"
	e) Emphasized Text	<em>			= digunakan untuk menunjukan bagian kata yang perlu ditekankan, kata yg ditekan akan miring.
	f) Importaant Text	<strong>		= digunakan untuk menunjukan sebuah teks bahwa kata/kalimat tersebut penting, biasanya akan berjetak tebal.
	g) Short Quations	<q>			= untuk menandai sebuah kupitan pada sebuah text.standarnya kalimat akan tampil dalam tanda kutip
	h) Citation 		<cite>			= selain sebagai sebuah atribut, <cite> juga merupakan seubah element yang digunakan untuk sebuah rusujuakn pada sebuah dokumen
	i) Defiining Terms	<dfn>			= 
	j) Subscript dan Superscript	<sub>|<sup>	= biasa digunakan untuk membuat teks menjadi tampilan kecil, seperti menunjukan seubah rumus kimia atau matematika
	k) highlighted Text	<mark>			= digunakan untuk menandai/menyorot(mewarnai) seeuah teks.
	l) Line Break		<br>			= untuk menambahkan baris baru
